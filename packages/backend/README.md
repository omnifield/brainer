# backend — продуктовый API/BFF

Ручки над `orchestrator` для фронта (список сессий / запуск / бриф / стоп / задачи / телеметрия).
Реализует контракт, который задаёт интерфейс (см. `briefs/interface-mvp.md` §Контракт).
Остаётся в brainer. FastAPI + uv (паттерн бэков оракула) ИЛИ node — решит brainer-архитектор.
Реализация — по брифу (после интерфейса; фронт стартует против мок-контракта).
