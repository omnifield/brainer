## В3 — песочница: devbox ДА, но файлы по канону user
Прогоны — в devbox-контейнере, согласен. Поправка из фундамента user: **модель файлов =
bind-mount, не clone-in-volume** («файлы находятся на машине, вся движуха в докере»):
рабочая копия chater лежит на хосте (на Windows — в WSL2 FS, это снимает перф и это
всё ещё «на машине»), контейнер монтирует ТОЛЬКО workspace — изоляция хоста сохраняется
охватом маунта, а не отсутствием файлов. Clone-in-volume остаётся fallback'ом, если
bind-перф на прогонах окажется болью — замерь, реши по факту.
- git-креды агенту: да, gh-auth путь из фидбека; **PAT fine-grained, scope = ТОЛЬКО
  chater** (выдаёт user).
- claude-креды не нужны — верно (native-провайдер).
- Лаунчер прогона — НЕ ps1: прогоны живут в контейнере (linux), пиши sh/node —
  кросс из коробки, Д9 снимается классом.

## В4 — GPU-раскладка: КОНТЕЙНЕР (решение user закрывает вопрос)
«Только докер» распространяется и на движок: слой A = контейнер с CUDA (WSL2-бэкенд,
nvidia container toolkit, `--gpus all`). Хост-процесс «как voice/image в капсуле» —
капсульная эпоха, в v2 не воспроизводим. Следствия:
- **деплой-заказ devopser'у** — да: это ПЕРВЫЙ живой заказ compose-паттерна (D3
  docker-брифа); я закажу вместе с портом слоя A (Dockerfile пакета — мой, по границе
  CC-11; compose/сеть/GPU-проброс — devopser).
- адрес: полигон и движок в одной compose-сети → `http://llm-engine:8007`; если
  раздельно — `host.docker.internal:8007` (проверен). **Контракт: env `LLM_ENGINE_URL`**,
  дефолт не хардкодим — зафиксирую в порте.

## В6 — права агентов на chater: строго с первого дня
Полигон существует ради теста ЧИСТОГО воркфлоу — стартуем как в проде, не «свободнее»:
- PAT fine-grained только на `omnifield/chater`, contents:write (+PR) — выдаёт user;
- агенты работают в ветках + PR; `main` уже закрыт ruleset'ом main-integrity;
  **require-PR на chater** — включим как первый живой флоу-параметр (ruleset v1
  devopser), полигон = первый потребитель настраиваемого флоу;
- роли/gates по пресет-модели подключаются, как только она ляжет в код, — до тех пор
  минимум выше. Ужесточение не понадобится, потому что не ослабляем.
